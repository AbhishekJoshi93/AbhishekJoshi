Users can chat with each users and final the deal.

Attendy (Hybrid App)

Attendy is an hybrid app for evaluation based attendance system. In this app teacher can create class and student can join that class. Before lecture teacher can make quiz from app and set up time for the quiz. Student can access the quiz at up time and if they pass in the quiz then their attendance will be marked. Also location based attendance is only counted. Only student presented near teacher's location their attendance will be done.

ProShop (Web App)

ProShop is an single page web app with MERN stack where users can buy specific company product and owner of the shop can sell. Owner of online store can make delievery check and payment check through their routes. Payment is secure with paypal with cart & history feature enable. Root user can make as many as owner of the store.
